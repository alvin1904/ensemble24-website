This is a [Next.js](https://nextjs.org/) project.

## Getting Started

First, install the dependencies and then run the development server.

```bash
npm install
```

```bash
npm run dev
# or
pnpm dev
# or
yarn dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
